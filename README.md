✨ 功能特性

📋 核心功能

· 网点清单管理 - 查看和管理所有网点信息
· 资费清单查看 - 显示详细的资费标准和价格信息
· 在柜信息查询 - 实时查看在柜物品的状态和位置
· 数据管理 - 添加、编辑、删除和搜索在柜数据
· 取件人信息管理 - 记录和管理取件人信息

🛠️ 管理功能

· 图片上传管理 - LOGO、背景、网点图片等
· 密码保护 - 敏感操作需要密码验证
· 本地存储 - 所有数据保存在浏览器本地
· 响应式设计 - 适配桌面和移动设备

🎯 用户体验

· 直观界面 - 标签页导航，操作简单
· 实时搜索 - 快速查找所需信息
· 图片放大查看 - 点击图片可全屏查看
· 状态指示器 - 清晰显示物品状态

🚀 快速开始

在线访问

将项目部署到任意静态网页服务器即可使用。

本地运行

1. 下载所有文件到本地
2. 双击打开 index.html 文件
3. 系统将自动在浏览器中打开

🔧 使用指南

1. 首次使用

1. 使用默认密码进入系统设置
2. 上传网点清单和资费清单图片
3. 配置系统LOGO和背景
4. 开始添加在柜数据

2. 数据管理

· 添加数据：点击"添加在柜数据"按钮
· 编辑数据：点击数据行右侧的编辑按钮
· 删除数据：点击数据行右侧的删除按钮
· 搜索数据：使用顶部搜索框

3. 图片管理

· 支持格式：JPG、JPEG、PNG
· 可上传：LOGO、背景、网点图、资费图等
· 点击图片可放大查看

4. 取件人信息

· 在"在柜信息"页面底部填写
· 需要姓名和联系电话
· 数据将保存在本地

💾 数据存储

本地存储位置

所有数据存储在浏览器的LocalStorage中：

· cabinetData - 在柜数据
· recipientData - 取件人信息
· logo / background / branch / price / contact - 图片数据

数据备份

建议定期导出数据备份：

1. 打开浏览器开发者工具
2. 进入Application > Local Storage
3. 复制相关数据保存
